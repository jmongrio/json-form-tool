# JSON Dynamic Form Tool

Aplicacion web para transformar un JSON en un formulario dinamico y mostrar la salida generada en tiempo real.

## Caracteristicas

- Entrada de JSON en texto para generar el formulario.
- Renderizado dinamico de campos segun el tipo de dato.
- Soporte para estructuras con objetos y arreglos.
- Panel de salida para visualizar el JSON resultante.
- Utilidades para detectar tipos JSON y mantener consistencia de datos.
